# FlatSat Lab

## Mission

Develop an educational FlatSat platform to learn:

- Satellite telemetry
- Flight software
- RF communications
- Ground station operations

## Hardware

- Raspberry Pi 4
- DHT22
- HC-12 (pending integration)

## Current Status

### Phase 1

- [x] Raspberry Pi OS Bookworm installed
- [x] Python environment configured
- [x] DHT22 connected successfully
- [x] Telemetry generation operational
